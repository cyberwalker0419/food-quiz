## 设计说明

- **向量值**: 基于菜品实际口感估算，范围 [0, 100]，符合归一化公式 `v = 50 + 50*k/m`
- **字段名**: 统一使用 `rich`(非 umami)，与 8 维体系一致
- **区域覆盖**: 覆盖所有省份/自治区/直辖市(除港澳)
- **特殊处理**: 傣菜保留独立(不并入云菜)，与云南汉族菜并存
- **无海外菜系**: 严格按中国大陆境内菜系
